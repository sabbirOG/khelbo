# KHELBO

**KHELBO** is a modern esports platform prototype. It features a neon, cyber-inspired design and is fully responsive for both desktop and mobile devices. While currently a prototype, it aims to become a full product soon, complete platform in the future.

## Features
- Esports-themed landing page
- Featured tournaments carousel
- Animated stats/HUD section
- Top players leaderboard
- Latest news marquee
- Featured clubs section
- Live video streams (YouTube embeds)
- Fully responsive (separate mobile.css)
